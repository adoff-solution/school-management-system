3. Borrowing & Returning Books Borrow Book: Allow students and faculty to borrow books. Key features: Assign a due date for return Limit the number of books a user can borrow at once Automatically update the number of available copies Return Book: Mark a book as returned, update the book's availability, and calculate any late fees (if applicable). Renew Borrowed Books: Allow users to extend the borrowing period for books if needed, if no one else has reserved them. Track Borrowing History: Keep a log of all books borrowed, including: Book name Borrowed date Due date Return date (if applicable) Fine (if the book is returned late)
